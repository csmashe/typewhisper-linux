# TypeWhisper for Linux v0.7.0

TypeWhisper for Linux 0.7.0 ports the major workflow, automation, provider, and
reliability updates from the upstream Windows branch where they apply to the
Linux desktop app.

## New Features

- Added batch file transcription queues with per-file status tracking and export
  options.
- Added watch folder transcription so folders can be monitored and processed
  automatically with configurable output.
- Added the recent transcriptions palette for quickly finding, editing, copying,
  and reusing recent dictation results.
- Added the ElevenLabs transcription plugin, including streaming support and
  localized settings.
- Added TTS provider plugin support with a Linux system voice provider backed by
  available desktop speech tools.
- Added local CLI installation/status controls in the Linux settings UI.

## Improvements

- Expanded the local HTTP API and bundled CLI with richer request parsing,
  status responses, translation targets, dictionary term management, and
  dictation controls.
- Improved automation compatibility for local tools through Raycast-compatible
  API behavior and CLI alignment.
- Improved file transcription handling for larger queues, media imports, and
  watch folder output.
- Improved plugin host behavior so bundled and external plugins can expose more
  capabilities consistently.
- Improved short and quiet speech handling with configurable padding and gain
  behavior.
- Hardened automatic paste on Linux by preferring clipboard paste, restoring the
  previous clipboard after successful insertion, and leaving the transcription on
  the clipboard when focus or paste fails.
- Added issue templates, a pull request template, a security policy, and more
  selective plugin smoke checks.

## Fixes

- Fixed dictation tail clipping so the end of short recordings is less likely to
  be cut off.
- Fixed prompt provider selection and related history/prompt provider race
  conditions.
- Hardened watch folder exception handling so background folder processing is
  more resilient.

## Not Ported From Windows

- Windows release channels and Velopack update-channel controls are not included
  in this Linux branch.
- Windows workflow automation and packaging changes remain Windows-specific.
